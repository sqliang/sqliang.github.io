# 开源洞察

::: note 说明：

本专栏所有源码解读文章，均借助了本人开源的 **[repo-insight](https://github.com/sqliang/repo-insight) Agent Skill** 来辅助生成。

**repo-insight** 是一款专注于优秀开源项目代码库解读的智能助手，提供 GitHub URL 或本地目录的一键深度分析能力。

* **一键分析** - 支持 GitHub URL 或本地目录分析
* **深度报告** - 生成项目概述、架构分析、核心模块解读
* **中文支持** - 自动翻译 README、生成中文代码注释
* **全自动执行** - 调用后无需中间交互

两个 Skill 均具备以上能力，可根据需求选择使用。
